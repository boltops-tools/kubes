## Usage

    kubes new variable

## Description

Generates kubes variable file.

## Examples

    $ kubes new variable
          create  .kubes/variables/dev.rb
    $ KUBES_ENV=prod kubes new variable
          create  .kubes/variables/prod.rb


## Options

```
y, [--force]  # Bypass overwrite are you sure prompt for existing files
```
